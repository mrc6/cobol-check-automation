# cobol-check-automation-
Automating COBOL testing with cobolcheck, zowe-cli and GitHub Actions.   

Learn about cobolcheck here https://github.com/openmainframeproject/cobol-check/wiki   

# How it works:   
When you submit a new code using this framework a github actions check if the tests are passing and upload your code   
from your repository to Mainframe  
   
# You need to configure the github actions secrets   
ZOWE_HOST The IP to access Mainframe   
ZOWE_PORT The PORT to access the Mainframe   
ZOWE_PASSWORD Your ZOS/Zowe password   
ZOWE_USERNAME Your ZOS/Zowe user
